# Urpak Edu — Авторизация және Көпқадамдық Тіркелу Жүйесі (Auth System)

Бұл жоба **Urpak Edu** интеллектуалды білім беру платформасының ашық (Light Mode) дизайн жүйесіне негізделген, пайдаланушыға ыңғайлы (Premium UX) Кіру және Көпқадамдық Тіркелу (Multi-step Registration Wizard) парақшасы.

## ✨ Ерекшеліктері (Features)

- **Бірыңғай Дизайн Жүйесі:** Платформаның негізгі Dashboard стилімен толық сәйкес келетін CSS tokens (`--indigo`, `--bg`, `--card`, `--ink`) қолданылған.
- **Динамикалық форма ауыстырғыш:** Бетті қайта жүктемей-ақ (без перезагрузки страницы) JavaScript арқылы Кіру, Тіркелу және Құпия сөзді қалпына келтіру формаларына өту.
- **Интерактивті 4 Қадамдық Тіркелу:**
  - **1-қадам (Рөл таңдау):** Оқушы, Мұғалім немесе Ата-ана рөлдерін көрнекі карточкалар арқылы таңдау (Оқушы рөлі әдепкі бойынша белсенді).
  - **2-қадам (Жеке деректер):** Пайдаланушының аты-жөні мен туған күнін енгізу.
  - **3-қадам (Контактілер):** Телефон нөмірін енгізу және СМС код сұрау.
  - **4-қадам (Растау):** Автоматты фокусталатын 4 таңбалы OTP код өрістері және 45 секундтық кері санақ (Countdown timer).
- **Сәтті аяқтау экраны (Success Screen):** Тіркелу толық біткенде "Перейти в кабинет" батырмасы бар арнайы анимациялық парақша шығады.
- **Backend-ке дайындық:** Барлық `<input>` және `<select>` тегтеріне Flask/Django немесе NodeJS базаларымен жұмыс істеуге арналған `name` атрибуттары (`name="role"`, `name="fullname"`, т.б.) алдын ала жазылған.

## 🚀 Жобаны локальді іске қосу (How to Run)

Бұл Frontend бөлімі таза **HTML5, CSS3 және Vanilla JavaScript** арқылы жазылғандықтан, оны іске қосу үшін қосымша серверлерді орнатудың қажеті жоқ.

1. Жобаны компьютеріңізге жүктеңіз (немесе `git clone` жасаңыз).
2. Жоба папкасындағы `index.html` файлын кез келген заманауи браузерде (Chrome, Safari, Firefox, Edge) екі рет басып ашыңыз.
3. Немесе VS Code бағдарламасында **Live Server** плагині арқылы іске қосыңыз.
